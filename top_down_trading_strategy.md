# トップダウン・トレード戦略：解説

このドキュメントは、市場の文脈（コンテキスト）を重視したトップダウン・アプローチによるトレード手法についてまとめたものです。

## 1. コア・コンセプト
**「個別銘柄は真空状態で動くのではない。全体の環境の一部として動く」**

どれほど完璧に見えるチャート設定（個別銘柄）であっても、市場全体の地合いや所属するセクターが逆風であれば、成功する確率は大幅に低下します。環境に逆らわず、追い風の中でトレードすることが重要です。

## 2. トップダウン・アプローチの3段階
分析は常に「広い視野」から「狭い視野」へと進めます。

1.  **市場全体 (Market Context)**
    *   $SPY (S&P 500) や $QQQ (Nasdaq 100) の動向を確認。
    *   主要な移動平均線の上にあるか、下にあるか？
    *   価格と出来高の振る舞いは健全か？

2.  **セクター/グループ (Sector/Group)**
    *   資金がどこに流れているか（キャピタル・フロー）を特定。
    *   そのグループは市場全体に対して「相対的な強さ」を示しているか？
    *   グループ内のリーダー銘柄（例: $NVDA など）はサポートを維持しているか？

3.  **個別銘柄 (Individual Stock)**
    *   市場とセクターのサポートがあることを確認した上で、ようやく個別銘柄のセットアップ（形）に焦点を当てる。

## 3. エントリー前のチェックリスト
ポジションを持つ前に、以下の質問を自分に投げかけます。

- [ ] 市場指数（$SPY, $QQQ）はサポートされているか？
- [ ] 市場は移動平均線の上にあるか？
- [ ] ターゲット銘柄のセクターは相対的に強いか、あるいは弱さを見せているか？
- [ ] セクター内のリーダー銘柄はサポートを維持しているか？
- [ ] **結論：なぜ今、このトレードを無理にする必要があるのか？（環境は味方しているか？）**

## 4. リーディングセクターの探し方 (Finviz 活用法)
複雑なツールは不要です。シンプルなステップで資金の流れを追います。

1.  **[Finviz](https://finviz.com/)** にアクセス。
2.  **Groups** → **Industry** をクリック。
3.  **1週間 (1 Week)** または **1ヶ月 (1 Month)** のパフォーマンスを確認。
4.  パフォーマンス上位のセクターをウォッチリストに入れ、その中の強い銘柄を精査する。

## 5. まとめ
「この株は買いか？」と尋ねる前に、**「市場とセクターはこの株をサポートしているか？」**を問うべきです。強さは強さを引きつけ、資金は単一の株ではなくグループ（テーマ）に流れ込みます。
